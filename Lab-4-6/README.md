Звіт про виконання лабораторної роботи №4-6

1. Мета роботи

Метою даної роботи було опанування практичних навичок роботи з мовою програмування JavaScript, зокрема:

Маніпуляція деревом DOM (Document Object Model).

Обробка подій (Event Handling) у браузері.

Робота з масивами та об'єктами даних.

Організація взаємодії між логікою скрипта та інтерфейсом користувача.

2. Опис функціоналу

Розроблений веб-додаток дозволяє користувачу взаємодіяти з динамічним інтерфейсом. Основні функції включають:

Введення даних: Форми для збору інформації від користувача.

Динамічне оновлення: Додавання або зміна елементів сторінки без її повного перезавантаження.

Обчислювальна логіка: Обробка введених значень за допомогою функцій JavaScript та виведення результату.

Стилізація: Використання CSS для забезпечення візуальної ієрархії та зручності користування.

3. Технічний стек

Для реалізації проєкту були використані наступні технології:

HTML5: Структурування контенту та створення семантичної розмітки.

CSS3: Оформлення зовнішнього вигляду, робота з макетом та позиціонуванням.

JavaScript (ES6+): Реалізація інтерактивності, логіки та управління станом сторінки.

GitHub Pages: Розгортання (deployment) проєкту для публічного доступу.

4. Аналіз структури проєкту

Проєкт організований згідно з принципом розділення відповідальності:

index.html — містить каркас додатку та підключення зовнішніх ресурсів.

style.css — описує візуальні правила.

script.js — містить основний програмний код.

5. Результати та висновки

В ході виконання роботи було успішно реалізовано алгоритми взаємодії з користувачем. Програма коректно обробляє вхідні дані та відображає їх у DOM-структурі.

Досягнуті показники:

Код є модульним та структурованим.

Веб-сторінка відповідає базовим вимогам UI/UX.

Реалізовано ефективний механізм прослуховування подій.
